# 项目简介

## 核心功能
- 基于YOLOv8的折断树木障碍检测系统
- 支持图片、视频及摄像头进行检测
- 支持选择不同检测模型
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 可选择是否显示标签和原图
- 提供多种数据输入方式：单张图片、图片文件夹、视频文件、本地摄像头

## 适用场景
- 森林资源监测与管理
- 林业安全巡查
- 自然灾害后的障碍物检测
- 野外环境中的道路安全评估

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 依赖Python 3.9及相关库：opencv-python、PyQt5、ultralytics、torch1.9等
- 包含1500张标注数据集，专注于['tree']一类目标
- 提供完整源码、训练好的模型、数据集及运行教程
- 配置成功环境后可直接使用
